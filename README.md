## 🏗️ Архитектура

- 📄 Экранная структура по страницам (`screen1`-`screen4`)
- 🧠 Бизнес-логика в composables
- 🧱 Переиспользуемые UI-блоки в components
- 🧭 Глобальный middleware для управления пользовательским потоком
